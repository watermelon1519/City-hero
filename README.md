# 🔪 城市英雄——三人小队卡牌战斗原型

当前仓库已经从最初的「背包英雄」演变为一款**三人小队 + 手牌连击结算**的战斗原型，用来验证核心战斗爽感。双击 `index.html` 即可在浏览器中游玩。

## 🎮 当前战斗玩法

### 队伍与敌人

- **我方三人小队**：  
  - 战士：高血量，可叠护盾挡伤害。  
  - 法师：高伤害技能，生命中等。  
  - 游侠：远程输出，后续可偏暴击/多段。  
- **敌人**：  
  - 顶部右侧显示敌人信息与血条，目前用于长时间测试，基础血量为 `1000`。  
  - 每回合会有「下回合意图」（例如：重击 15 伤害）。

### 手牌与过牌

- 每回合开始时，自动从牌库抽取 **5 张手牌**。  
- **单击手牌**：标记/取消为「要换掉的牌」（卡牌金色描边）。  
- **点击「换牌」按钮**：  
  - 将所有标记的手牌丢入弃牌堆；  
  - 从牌库重新抽取同样数量的新牌（牌库空时会把弃牌堆洗回牌库继续抽）；  
  - 每回合只能换牌一次。  
- 牌库/弃牌堆数量会显示在按钮上方，并可展开右侧面板查看弃牌堆的具体牌。

### 结束回合 = 出牌 + 结算

- 点击「结束回合」时，本回合的 5 张手牌会统一结算：
  1. **治疗 & 护盾先结算**（见下一节）。  
  2. 将仍在手里的所有攻击牌按职业分布传入牌型系统，计算：  
     - 对子 / 三条 / 两对 / 葫芦 / 四条 / 同花 / 顺子 / 同花顺 等牌型倍数；  
     - 职业组合技（如战+法=魔法剑、战+法+游=铁三角 等）；  
     - 最终本回合总伤害，并一次性打在敌人身上。  
  3. 所有手牌进入弃牌堆，敌人进行一次攻击，然后自动抽新 5 张进入下一回合。  
- 顶部中间会显示「**连击倍数 × 本回合伤害**」，战斗日志会记录牌型和组合技详情。

### 治疗与目标绑定（拖拽）

- **单体治疗牌**（例如：`治疗术`、`药水`）：  
  - 这类牌具有 `heal` 属性，且没有 `healAll`。  
  - 从手牌中**拖拽该牌到上方某个队友头像上**（战士/法师/游侠）即可：  
    - 为这张牌绑定一个独立目标；  
    - 该牌卡框会变成绿色（已绑定目标）；  
    - 被绑定的角色头像右上角会出现计数标记：  
      - 💚 + 数字：本回合指向他的单体治疗术数量；  
      - 🧪 + 数字：本回合指向他的药水数量。  
  - 每张牌可以绑定不同目标，多张治疗可以分别给不同人加血。  
- **群体治疗牌**（例如：`群体治疗`）：  
  - 具有 `healAll` 属性，不需要绑定目标，会对所有存活队友生效。  
- **回合结束前校验**：  
  - 如果手牌中还有**未绑定目标的单体治疗牌**，点击「结束回合」时会阻止结算，并在日志提示「还有治疗牌未分配目标」。  
  - 只有所有单体治疗都拖拽完并绑定了目标，才能正常结束回合。

### 护盾与格挡

- 每个角色都有一个独立的**护盾值**，显示在血量下方：`🛡️ 8`。  
- 当前所有带 `shield` 属性的牌（例如 `格挡`）会为战士叠加护盾：  
  - 护盾可以多次叠加，不会自动清零。  
- 敌人攻击时的伤害顺序：  
  1. 先从被选中角色的护盾中扣除伤害（护盾值减少）；  
  2. 若还有剩余伤害，再对生命值造成伤害；  
  3. 日志会显示「战士的护盾吸收了 X 点伤害」。  
- 后续可以为某些敌人技能增加「毒伤害 / 穿透」来无视护盾。

## 🚀 快速开始

### 方法一：直接打开

在文件管理器中双击 `index.html`，使用现代浏览器（建议 Chrome / Edge / Safari）打开即可。

### 方法二：本地静态服务器

```bash
# 在项目根目录下

# 方式 1：Python 3
python -m http.server 8000

# 方式 2：Node.js（需要全局或本地安装 http-server）
npx http-server -p 8000
```

然后在浏览器访问 `http://localhost:8000`。

## 📝 操作说明

- **选择治疗目标**：  
  - 从手牌中拖拽 `治疗术` / `药水` 到上方某个队友头像上，完成牌→目标绑定。  
  - 角色头像右上角会出现 💚/🧪 标记及数量。  
- **标记换牌**：  
  - 单击任意手牌，使其金色描边，点击「换牌」会丢弃这些牌并重抽。  
- **结束回合**：  
  - 在所有治疗牌都绑定好目标后点击「结束回合」，系统会先结算治疗/护盾，再按牌型一次性结算伤害，然后进入敌人回合。  
- **查看弃牌堆**：  
  - 点击「弃牌堆」数字或「查看弃牌堆」下拉，即可看到本局已打出的牌。
